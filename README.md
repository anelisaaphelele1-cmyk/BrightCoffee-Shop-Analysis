# BrightCoffee-Shop-Analysis
**BrightCoffee-Shop-Analysis**
A comprehensive, data-driven exploration of coffee sales performance, quality, and roasting characteristics using SQL. This repository presents an in-depth analysis of sales data from a coffee shop chain operating across three branches, covering transactions from January to the present. The project uncovers performance patterns, customer preferences, and product-level insights through data analytics and visualization.
